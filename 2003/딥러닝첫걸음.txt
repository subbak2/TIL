딥러닝첫걸음
 - 김성필 지음, 한빛미디어

1. 용어 정리
 - 인공지능 > 머신러닝 > 딥러닝 (포함관계)
 - 머신러닝 : 데이터를 통한 모델링 기법
                  지도학습 = {입력, 정답}
                  비지도학습 = {입력}
                  강화학습 = {입력, 출력, 점수}

2. 신경망
 - 노드들간의 관계와 가중치, 계층을 배치한 네트워크, 입력망, 은닉충, 출력층
 - 일반식 : v  = w1*x1+w2*x2+w3*x3 + b
 - 델타규칙 : 경사하강법의 일종, 임의의 초기값 -> 단계적 정답
 - SGD : 한 개의 학습 데이터를 매번 가중치 갱신 -> 무작위 학습 가능
 - 배치 : 오차 -> 가중치 갱신값을 계산 평균값으로 갱신
 - 미니배치 : 데이터 개수로 나누어 배치
 - 역전파 알고리즘 : 역순으로 오차를 정의하면서 해결 (아직 잘 모르겠음)

3. 딥러닝 
 - 은닉충이 2개 이상인 심층 신경망
 - 은닉충 1개 추가의 의미
   1) 역전파 알고리즘으로 오차 계산 해결
   2) 성능 저하
      - 그레디언트 소실 -> ReLU 함수로
      - 과적합 -> 드롭아웃을 통해 과적합 감소
      - 많은 계산 -> 고성능 하드웨어&배치 정규화

4. 컨벌루션 신경망
 - 영상인식에 특화된 심층 신경망, 풀링 계층을 통해 입력 이미지에서 픽셀을 묶어 대표값으로 인식
